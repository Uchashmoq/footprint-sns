# 🌍 足迹 Footprint

加入我们：📮`72e1bcgg@gmail.com` Ma Haoran

**足迹**是一款为 **旅行者、摄影师、户外记录者** 打造的轻社交平台。
 在这里，你可以分享照片、记录位置、讲述你的旅途故事，也可以发现和你一样热爱世界的朋友。

> 📸 带上相机，背起背包，让你的足迹被看见。

------

## 🚀 技术栈

Next.js · Tailwind CSS · DaisyUI · Prisma · react-leaflet · Clerk

------

## ✨ 功能特性 Features

------

## 📸 1. 发布并浏览旅拍作品

上传照片、添加描述、附带定位，就可以生成一条“足迹”。
 如果你是专业摄影师，你还可以上传 **RAW 文件** 并展示 EXIF 信息，甚至允许其他用户 **付费下载 RAW**。

其他用户可进行：

- 👍 点赞
- 💬 评论
- ⬇️ 下载 RAW
- 👤 访问你的主页

## 🗺️ 2. 在地图上探索全球足迹

通过世界地图，你可以查看全球真实用户留下的足迹：
 某个城市、某个海岸、某座山…看看那里曾发生什么。

还能打开用户主页，查看他的旅行轨迹与拍摄位置。

## 🗂️ 3. 照片集 / 相册系统

用户可以创建主题相册，例如：

- 城市街拍
- 山野徒步
- 海岸线
- 夜景摄影

让旅行作品更有组织，也更便于展示。

## 🤝 4. AI 兴趣匹配（开发中）

未来将加入 AI，根据以下信息推荐与你兴趣相似的人：

- 拍摄地点
- 照片内容
- 行走轨迹
- 旅行偏好

让热爱旅行和摄影的人更容易连接彼此。
